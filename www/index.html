Styling der Komponente

Globale Definitionen um alle Komponenten identisch zu stylen

  /* <style> set all speaker at page to defaults of component via :root */
    :root {
      --honey-news-background: transparent;
      --honey-news-color: blue;
      --honey-news-padding: 5px;
      --honey-news-font-size: medium;
      --honey-news-border: 0;
      --honey-news-width: 36px;
      --honey-news-height: 36px;
    }

Styling über css class und custom properties verändert

  # part of style sheet

    #css1 {
      --honey-news-color: gray;
    }

    .css2class {
      --honey-news-color: red;
      --honey-news-background: #E8CD9A;
    }

    #css3 {
      --honey-news-color: green;
    }
    .css3class {
      --honey-news-background: #E8CD9A;
    }

  # part of html

    <honey-news id="css1" textids="artikel2"></honey-news>
    <honey-news textids="artikel2" class="css2class"></honey-news>
    <honey-news id="css3" textids="artikel2" class="css3class"></honey-news>
